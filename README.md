## 🚀 Como Executar Localmente


1. Certifique-se de ter instalado:
   - Node.js (v18+)
   - Angular CLI (V19)
   - Git

2. Clone o repositório:  
   ```bash
   git clone https://github.com/seu-usuario/portfolio.git
   ```

3. Instale as dependências:
    ```bash
      npm install
    ```

 4. Inicie o servidor
      ```bash
      ng serve
      ```
5. Acesse:
      ```bash
      http://localhost:4200
      ```
